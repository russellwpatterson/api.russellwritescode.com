Alarm 3.5.x Release Notes

3.5.18 (July 7, 2011)

  • Change: Added ability to specify alarm time by link. Use this format: "alarm.exe /s19:30"
  • Change: Moved color scheme dialog into main Options window. Didn't make sense elsewhere.
  • Change: Option to ask to confirm alarm reset, snooze, and silence.
  • Change: Can now save and load alarm times to .alarm files.
  • Bug Fix: Tray Icon flashes on startup.

3.5.17 (February 8, 2011)

  • Change: Added confirmation dialog when resetting the alarm time via the button.
  • Change: Changed "Alarm Off" to "Reset"

3.5.16 (December 20, 2010)

  • Bug Fix: Issue when 24 hour clock is used with Snooze or Quick Alarm.
  • Change: Rewrote clock and alarm functionality to always store 24-hour times.

3.5.15 (November 9, 2010)

  • Bug Fix: Problems when "Remember Screen Location" and "Minimize to Tray" are enabled together.

3.5.14 (June 29, 2010)

  • Bug Fix: Playlist didn't advance to the next song while going off.

3.5.12 (June 23, 2010)

  • Bug Fix: Color problem on Windows XP Classic theme
  • Change: Beep Audio file
  • Addition: WAVs can now be used in the playlist
  • Addition: 24 Hour Clock option
  • Change: Media is now played through Windows Media Player API. This should fix all issues with playback.
  • Removed: Retain Focus option. Too many complaints.
  • Addition: Safe Mode for Antivirus software
  • Change: "Media" tab in Options is now the "Sound" tab
  • Addition: 5 Minute Quick Alarm
  • Addition: Remember Screen Location option
  • Change: Massive refactoring throughout the program to increase efficiency
  • Bug Fix: Focus Alarm Time + Snooze = UI Issue

3.5.11 (December 23, 2009)

  • Addition: 2,3,4,6,8 Hour and Custom Quick Alarms. Thanks to Missy for the suggestion.
  • Addition: Alarm 4 About dialog added.
  • Change: Removed one of the first startup dialogs. (The thanks dialog)
  • Change: Code refactoring
  • Change: Automatically Displays Morning Reminder when Alarm goes off.
  • Change: Some changes to the default settings.
  • Addition: Release Notes Viewer
  • Change: Check for Updates doesn't run on first startup.
  • Addition: University of Akron (Grad School) Color Scheme.
  • Change: Adjust Volume now disabled when media isn't selected.
  • Change: Windows 7 compatibility change.

3.5.10 (August 3, 2009)

  • Bug Fix: Fixed a bug in the app launcher.
  • Change: Changed the way "Prevent Windows Update Restart" is done.
  • Change: Compiled under Visual Studio 2008 for first time.

3.5.9 (July 3, 2009)

  • Change: Update has been changed to prepare for Alarm 4 updating.
  • Addition: Ability to set volume. This only applies to the application so you can have your system volume at whatever you want. A special thanks to JJ Fullmer for the suggestion.
  • Addition: Ability to choose whether or not the alarm time takes the place of the time when the alarm is set. Thanks to "OldeFart" for the suggestion.

3.5.8 (May 18, 2009)

  • Addition: Ability to Launch an Application when the alarm goes off.
  • Change: The Automatic Update Feature has now been updated to AUA v4.2 which is a completely re-written system which future versions will use.
  • Addition: New "Alarm Off" button when the alarm is set, but not going off.
  • Change: "Turn Off" has been renamed to "Silence"
  • Change: Internally capable of using a product activation system. However, it is not active at this time.
  • Change: Loads of refactoring done.
  • Bug Fix: Stupid little issue with automatically turning on certain settings. Ugh.
  • Addition: Option to make the Alarm retain focus while it's going off. This is a very good feature if you're using a media center remote to hit snooze in the morning. The focus stops when the mouse moves over the main window.

3.5.7 (April 5, 2009)

  • Bug Fix: Some songs aren't properly faded in.
  • Change: Playlist now asks to confirm clearing it.

  • Change: Updated Russell's, Crimson Clock, Dark Night, Ohio State(Now Scarlet and Grey), and Purple Haze color schemes to be better on the eyes. Ohio Northern is now called "Alma Mater" since I'm going to be graduating here in couple months.
